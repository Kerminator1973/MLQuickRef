# Преобразование объектов DataFrame

В действительности, раздел касается обработки данных с любыми таблицами (CSV, реляционные СУБД, и т.д.).

При подготовке данных для использования часто используется т.н. "изменение формы" (resharping), т.е. приведение данных в другой вид.

Терминологически, данные могут храниться в широком (wide), или узком фомате (narrow, long, tall).

Пример широкого представления:

|   | Weekday | Miami | New York | Chicago | San Francisco |
|---|---------|-------|----------|---------|---------------|
| 0 | Monday  |  100  |    65    |   50    |      60       |
| 1 | Tuesday |  105  |    70    |   58    |      62       |

Широкий набор данных идеально подходит для изучаения общей картины происходящего. Однако по мере добавления новых столбцов работать с данными становится всё сложнее.

Узкий формат значительно упрощает обработку данных. Пример трансформации данных:

|   | Weekday | City          | Temperature |
|---|---------|---------------|-------------|
| 0 | Monday  | Miami         | 100         |
| 1 | Monday  | New York      | 65          |
| 2 | Monday  | Chicago       | 50          |
| 3 | Monday  | San Francisco | 60          |
| 4 | Tuesday | Miami         | 105         |
| 5 | Tuesday | New York      | 70          |

## Агрегация

В Pandas агрегацию часто называют "созданием сводной таблицы" (pivot-tables). Цитата из книжки: "Слово "агрегирует" означает вычисление сводных показателей на основе нескольких значений".

Пример создания сводной таблицы в Pandas:

- Выбираем столбцы, значения которых мы хотим агрерировать
- Выбираем применяемую к столбцам операцию агрегирования
- Выбираем столбцы, на основе значений которых будет группировать агрегированные данных по категориям
- Выбираем, где размещать группы - на оси строк, оси столбцов или на обеих осях координат

Типовой пример аггрегации по значению поля "Date":

```py
sales.pivot_table(index = "Date")
```

Указать агрегирующую функцию можно с помощью параметра aggfunc:

```py
sales.pivot_table(index = "Date", aggfunc = "mean")
```

Мы можем указать только один столбец, по которому следует выполнять агрегацию. Например, мы можем суммировать данные только по полю "Revenue":

```py
sales.pivot_table(index = "Date", values = "Revenue", aggfunc = "sum")
```

Pandas позволяет выполнять разные операции агрерации над разными полями:

```py
sales.pivot_table(
    index = "Date",
    columns = "Name",
    values = ["Revenue", "Expenses"],
    fill_value = 0,
    aggfunc = {"Revenue": "min", "Expenses": "max"}
)
```

Иногда бывает нужно перенести уровень индекса с одной оси на другую, представить данные в другом виде и затем выбирать, какое представление нам лучше подходит.

Метод stack() переносит уровень индекса с оси столбцов на ось строк. Пример вызова:

```py
by_name_and_date.stack().head(7)
```

Дополняющий stack() метод unstack() переносит уровень индекса с оси строк на ось столбцов.

## Широкие и узкие данные. "Расплавление" (Melting) набора данных

Сначала о том, что такое широкие и узкие данные...

Пример широких данных:

| Date       | Name  | Customer      | Revenue | Expenses |
|------------|-------|---------------|---------|----------|
| 2020-01-01 | Oscar | Logistics XYZ | 5250    | 531      |

Пример узких данных - каждое значение строки соответствует отдельному наблюдению заданной переменной:

| Date       | Category | Value         |
|------------|----------|---------------|
| 2020-01-01 | Name     | Oscar         |
| 2020-01-01 | Customer | Logistics XYZ |
| 2020-01-01 | Revenue  | 5250          |
| 2020-01-01 | Expenses | 531           |

Широкие данные удобнее анализировать визуально, а узкие данные гораздо удобнее агрегировать.

Расплавление данных - это обратная операция созданию сводной таблицы. Для выполнения расплавления (Melting) в Pandas используется специальная функция melt():

```py
video_games_sales.melt(id_vars = "Name", value_vars = "NA").head()
```

Один из типовых примеров обработки данных:

- Сначала разобрать строку отдельной ячейки, в которой перечисляются элементы
- Затем развернуть значения из списка в несколько строк

Сделать это можно, приблизительно, следующим образом:

```py
recipes["ingredients"] = recipes["ingredients"].str.split(",")
recipes.explode("ingredients")
```

## Группировка

Предположим, что у нас есть следующие данные:

```py
food_data = {
    "Item": ["Banana", "Cucumber", "Orange", "Tomato", "Watermelon"],
    "Type": ["Fruit", "Vegetable", "Fruit", "Vegetable", "Fruit"],
    "Price": [0.99, 1.25, 0.25, 0.33, 3.00]
}

supermarket = pd.DataFrame(data = food_data)
```

Мы можем выполнить группировку по имени группы, например, по "Type" (овощ, или фрукт):

```py
groups = supermarket.groupby("Type")
```

Затем мы можем получить значения (DataFrame) только нужной нам группы:

```py
groups.get_group("Fruit")
groups.get_group("Vegetable")
```

Объект groups удобен сам по себе, т.к. он повзоляет выполнять агрегацию по группам, например:

```py
groups.mean()
```
