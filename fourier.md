# Преобразование Фурье

Посредством преобразования Фурье осуществляется разложение исходной функции на элементарные составляющие — гармонические колебания с разными частотами. Под разложением подразумевается, что результатом работы является другая функция, которая описывает коэффициенты ("амплитуды") при разложении исходной функции на элементарные составляющие. В терминах языков программирования, результат может быть представлен в виде массива коэффициентов.

Рекомендуется для ознакомления статья [Простыми словами о преобразовании Фурье](https://habr.com/ru/articles/196374/) by 0serg.

Если приводить аналогии, то примером аналогичного по смыслу преобразования может послужить дифференцирование, превращающее функцию в её производную. Преобразование Фурье — такая же, по сути, операция как и взятие производной, только в отличие от дифференцирования, преобразование Фурье всегда "работает" с более общими комплексными числами. Из-за этого постоянно возникают проблемы с отображением результатов этого преобразования, поскольку комплексные числа определяются не одной, а двумя координатами на оперирующем действительными числами графике.

В графическом представлении используется два графика: 

- график аргумента комплексного значения часто называют в "фазовым спектром"
- график модуля — "амплитудным спектром"

Амплитудный спектр как правило представляет намного больший интерес, а потому “фазовую” часть спектра нередко пропускают.

Исключение не существенных деталей, позволяет достичь следующих целей:

- снижения размера данных, описывающих сложную функцию (сжатие)
- исключение малозначимых деталей/шума, с целью повысить точность сравнения двух функций

Чаще всего используется быстрое преобразование Фурье (FFT).

## Применение для 2D-графики

Преобразование Фурье можно использовать и для работы с 2D-графикой. В этом случае, часто рассматривают изображение как функцию, в которой координата Y - геометрическая позиция точки, а Y - интенсивность точки.
