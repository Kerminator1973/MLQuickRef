# Векторизация текстовых сообщений

Векторизация в **Pandas** — это процесс преобразования текстовых данных в числовые представления, что позволяет эффективно использовать алгебраические и статистические операции. Этот процесс обеспечивает преобразование текстов в формат, который можно использовать в машинном обучении и анализе данных.

Методы векторизации:

- **Count Vectorization**: Превращает текст в векторы, где каждый элемент соответствует количеству вхождений слова.
- **TF-IDF** (Term Frequency-Inverse Document Frequency): Оценивает важность слова в документе, учитывая, как часто оно встречается в других документах
- **Word Embeddings**: Использует предварительно обученные модели, такие как Word2Vec или GloVe, для представления слов в многомерном пространстве

Преимущества векторизации

- Позволяет осуществлять количественный анализ
- Улучшает обработку текстов в машинном обучении
- Упрощает сравнение текстовых данных

Векторизация — ключевой этап подготовки текстовых данных для последующего анализа и моделирования.

Принципы формирования векторов

## Создание словаря

В процессе векторизации создается словарь, который содержит все уникальные слова (или токены) **из коллекции документов**. Каждое слово получает свой уникальный индекс.

Формирование векторов:

- Векторы формируются для каждого документа. Элементы вектора могут представлять:
- Количество вхождений каждого слова (в случае Count Vectorization).
- Важность слова с учетом его частоты и распределения по документам (в случае TF-IDF).

### Пример

Если у вас есть три документа:

- Документ 1: "Кошка на дереве"
- Документ 2: "Собака на траве"
- Документ 3: "Кошка и собака"

Создается словарь:

- "Кошка" (индекс 0)
- "на" (индекс 1)
- "дереве" (индекс 2)
- "Собака" (индекс 3)
- "траве" (индекс 4)
- "и" (индекс 5)

Векторы документов будут выглядеть следующим образом:

- Документ 1: [1, 1, 1, 0, 0, 0]
- Документ 2: [0, 1, 0, 1, 1, 0]
- Документ 3: [1, 0, 0, 1, 0, 1]

Таким образом, каждый документ представлен в виде числового вектора, что позволяет применять математические и алгоритмические методы анализа.

### Основные типы данных для векторов

- **NumPy Array**: Чаще всего векторы представлены как объекты типа numpy.ndarray. Это позволяет эффективно выполнять математические операции и использовать функции линейной алгебры
- **Pandas Series**: Векторы также могут быть представлены как pandas.Series, что позволяет удобно взаимодействовать с ними и использовать все преимущества функциональности библиотеки Pandas.
- **SciPy Sparse Matrices**: В случае больших наборов данных, где много элементов векторов равны нулю, используются разреженные матрицы (scipy.sparse), чтобы экономить память и ресурсы. Это особенно полезно при использовании методов, таких как Count Vectorization и TF-IDF.
