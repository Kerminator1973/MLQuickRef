# scikit-learn

Официальный сайт [scikit-learn](https://scikit-learn.org/stable/)

Содержит большое количество разных моделей, включая линейную регрессию. Также содержит методы оценки качества, включая mse. Реализую методы разделения данных на выборки для обучения и выборку для валидации.

Типовой код:

```python
from sklearn.linear_model import LinearRegression
# Извлекаем какие-то данные из Dataset-а, который можно, например, скачать можно используя sklearn.datasets
# Присваиваем данные: X, y = ...
model = LinearRegression(fit_intercept=True)
model.fit(X, y)
prediction = model.predict(X)
```

В X находятся признаки, а в y - правильные ответы. Обучение модели на загруженных данных осуществляется с помощью вызова `fit()`. Выполнение предсказания осуществляется посредством вызова `predict()`.

Т.е. нам нужно:

- импортировать модель
- загрузить данные для обучения
- создать экземпляр модели
- провести обучение модели на данных
- используем обученную модель для получения результата
