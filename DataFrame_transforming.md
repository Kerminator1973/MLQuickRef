# Преобразование объектов DataFrame

В действительности, раздел касается обработки данных с любыми таблицами (CSV, реляционные СУБД, и т.д.).

При подготовке данных для использования часто используется т.н. "изменение формы" (resharping), т.е. приведение данных в другой вид.

Терминологически, данные могут храниться в широком (wide), или узком фомате (narrow, long, tall).

Пример широкого представления:

|   | Weekday | Miami | New York | Chicago | San Francisco |
|---|---------|-------|----------|---------|---------------|
| 0 | Monday  |  100  |    65    |   50    |      60       |
| 1 | Tuesday |  105  |    70    |   58    |      62       |

Широкий набор данных идеально подходит для изучаения общей картины происходящего. Однако по мере добавления новых столбцов работать с данными становится всё сложнее.

Узкий формат значительно упрощает обработку данных. Пример трансформации данных:

|   | Weekday | City          | Temperature |
|---|---------|---------------|-------------|
| 0 | Monday  | Miami         | 100         |
| 1 | Monday  | New York      | 65          |
| 2 | Monday  | Chicago       | 50          |
| 3 | Monday  | San Francisco | 60          |
| 4 | Tuesday | Miami         | 105         |
| 5 | Tuesday | New York      | 70          |

## Агрегация

В Pandas агрегацию часто называют "созданием сводной таблицы" (pivot-tables). Цитата из книжки: "Слово "агрегирует" означает вычисление сводных показателей на основе нескольких значений".

Пример создания сводной таблицы в Pandas:

- Выбираем столбцы, значения которых мы хотим агрерировать
- ыбираем применяемую к столбцам операцию агрегирования
- Выбираем столбцы, на основе значений которых будет группировать агрегированные данных по категориям
- Выбираем, где размещать группы - на оси строк, оси столбцов или на обеих осях координат

Типовой пример аггрегации по значению поля "Date":

```py
sales.pivot_table(index = "Date")
```

Указать агрегирующую функцию можно с помощью параметра aggfunc:

```py
sales.pivot_table(index = "Date", aggfunc = "mean")
```
