# Визуализация

Для визуализации результатов обработки, чаще всего используются такие библиотеки, как: Matplotlib, seaborn и ggplot.

Matplot для отображения динамических диаграмм использует Series и DataFrame.

Установка библиотеки Matplot:

```shell
conda install matplotlib
```

## Линейный график

Допустим, что мы импортировали данных из CSV-файла:

```py
space = pd.read_csv(
    "space_missions.cvs",
    parse_dates = ["Date"],
    index_col = "Date"
)
```

Заметим, что у нас есть поле "Date", которое стало индексом - значения этого поля будут отображаться по оси X. Далее мы можем выбрать другое поле, значения которого будут проекцироваться на ось Y. Вывод графика:

```py
space["Cost"].plot()
```

Мы можем использовать DataFrame, состояший из нескольких колонок и отобразить несколько графиков на одном листе. Необходимо помнить, что индекс отображается по оси X, а значения разных колонок проекцироваться на ось Y.

Если нам необходимо использовать разные цвета для разных графиков, то мы можем использовать цветовую схему (colormap):

```py
space.plot(y = "Cost", colormap = "gray")
```

Список доступных цветовых схем можно посмотреть командой:

```py
print(pl.colormaps())
```
