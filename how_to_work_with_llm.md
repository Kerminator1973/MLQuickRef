# Как работать с большой языковой моделью

Данный документ создан на основе курса Нетологии "Принципы работы ИИ"

**Токен** — это минимальная значимая единица текста для модели.

Для запроса: «Объясни, что такое EBITDA простыми словами», роследовательность токенов может быть такой: «Объясни», «,», «что», «такое», «EB», «IT», «DA», «простыми», «словами».

Длинные и редкие термины, такие как EBITDA, почти всегда разбиваются на несколько токенов, потому что встречаются реже и сложнее для единого представления.

Контекстное окно = вход + сгенерированная часть ответа + последующий диалог. Если окно заполнено, старые фрагменты текста вытесняются.

>Как работать с контекстным окном:
>
>- Делите длинные запрос на смысловые части
>- Регулярно актуализируйте контекст: «Перечисли ключевые факты, которые ты учитываешь дальше»
>- Резюмируйте текст перед отправкой
>- Уточняйте лимит: «Сколько токенов ты можешь обработать во входе + ответе?»

Неполная задача для ИИ — это риск получить уверенный, но неправильный анализ, даже если задача очевидна человеку.

Чтобы нейросеть решила задачу корректно, запрос должен включать пять элементов: действие, объект, данные, параметры и формат ответа.

- Действие — что нужно сделать. Примеры действий: объясни, посчитай, сравни, проверь, структурируй, выдели выводы
- Объект — к чему относится задача. Нужно явно указать, что является фокусом анализа: выручка, расходы, категория, период, направление бизнеса
- Данные — на основе чего работать. В конце запроса всегда указывайте, где находятся данные: их можно скопировать прямо в сообщение или приложить файлом
- Параметры — в каких условиях выполнять задачу. Модели нужны границы: период, метод расчёта, единицы измерения, критерии сравнения
- Формат — как должен выглядеть результат. Формат определяет форму вывода: таблица, список, формула, короткое резюме

Чтобы минимизировать ошибки в ответе, проверьте понимание запроса одним из двух приёмов:

- Попросите нейросеть пересказать задачу: "Перескажи, что ты понял из моего запроса кратко, по пунктам"
- Попросите модель задать уточняющие вопросы: "Перед выполнением задачи задай уточняющие вопросы, которые помогут определить контекст и шаги"
